## 第一段

欢迎使用 Next.js 静态博客！

这是一个功能完整的静态博客系统，支持：

- Markdown 文章编写
- 明暗主题切换
- 文章目录（TOC）
- 系列和分类管理
- 无限滚动（首页列出所有文章）
- 404 页面
- 站点地图自动生成

## 第二段

支持 `code` 与 **加粗**，以及：

1. 有序列表
2. 无序列表
3. [链接](https://nextjs.org)
4. 代码块

```javascript
function hello() {
  console.log('Hello, World!');
}
```
